# нҢҖмӣҗлі„ кө¬нҳ„ кё°лҠҘ мғҒм„ё

### к°•м§Җмӣҗ

- **лҢ“кёҖ л“ұлЎқ**
    - лҰ¬л·°м—җ лҢ“кёҖмқ„ мһ‘м„ұн•ҳлҠ” кё°лҠҘ кө¬нҳ„
    - ReviewRepositoryмҷҖ м—°лҸҷн•ҳм—¬ л…јлҰ¬мӮӯм ңлҗң лҰ¬л·°м—җлҠ” лҢ“кёҖ л“ұлЎқ л¶Ҳк°Җ мІҳлҰ¬
    - Notificationкіј м—°лҸҷн•ҳм—¬ лҢ“кёҖ мһ‘м„ұ мӢң н•ҙлӢ№ лҰ¬л·° мһ‘м„ұмһҗм—җкІҢ м•ҢлҰјмқҙ к°ҖлҸ„лЎқ мІҳлҰ¬
    - лҢ“кёҖ л“ұлЎқ мӢң лҰ¬л·°мқҳ commentCount мҰүмӢң лҸҷкё°нҷ” мІҳлҰ¬
- **лҢ“кёҖ мҲҳм •**
    - ліёмқёмқҙ мһ‘м„ұн•ң лҢ“кёҖмқём§Җ кІҖмҰқ нӣ„ лҢ“кёҖмқ„ мҲҳм •н•ҳлҠ” кё°лҠҘ кө¬нҳ„
- **лҢ“кёҖ лӢЁкұҙ мЎ°нҡҢ**
    - л…јлҰ¬мӮӯм ңлҗҳм§Җ м•ҠмқҖ лҢ“кёҖмқ„ мЎ°нҡҢн•ҳлҠ” кё°лҠҘ кө¬нҳ„
- **лҢ“кёҖ лӘ©лЎқ мЎ°нҡҢ**
    - @QueryлҘј нҷңмҡ©н•ң м»Өм„ң нҺҳмқҙм§Җл„Өмқҙм…ҳ кё°л°ҳ лҢ“кёҖ лӘ©лЎқ мЎ°нҡҢ кё°лҠҘ кө¬нҳ„
    - createdAt, commentId л‘җ к°Җм§ҖлҘј ліөн•© м •л ¬ мЎ°кұҙмңјлЎң мӮ¬мҡ©н•ҳм—¬ м •нҷ•н•ң мҲңм„ңлҘј ліҙмһҘн•ҳлҸ„лЎқ кө¬нҳ„
    - AND/OR мҡ°м„ мҲңмң„ кҙ„нҳё мІҳлҰ¬лЎң лӢӨлҘё лҰ¬л·° лҢ“кёҖ нҳјмһ… л°©м§Җ
- **лҢ“кёҖ мӮӯм ң**
    - softDelete, hardDelete л‘җ к°Җм§Җ кё°лҠҘмқ„ лӘЁл‘җ м§Җмӣҗн•ҳлҸ„лЎқ кө¬нҳ„
    - softDeleteлҠ” deletedAt н•„л“ңлҘј кё°лЎқн•ҳм—¬ л…јлҰ¬мӮӯм ң мІҳлҰ¬
    - hardDeleteлҠ” DBм—җм„ң мҳҒкө¬мӮӯм ң мІҳлҰ¬
    - л…јлҰ¬мӮӯм ңлҗң лҢ“кёҖмқ„ л¬јлҰ¬мӮӯм ң мӢң commentCount мқҙмӨ‘ м°Ёк°җ л°©м§Җ лЎңм§Ғ м Ғмҡ©
- **кіөнҶө кё°м—¬**
    - BaseEntity лӮ м§ң нҳ•мӢқ ISO 8601 (yyyy-MM-dd'T'HH:mm:ss) нҶөмқј
    - м „ лҸ„л©”мқё мҳӨлҘҳ кІҪмҡ°мқҳ мҲҳ л¶„м„қ л°Ҹ нҢҖ л…ём…ҳ л¬ём„ңнҷ”

### к№ҖмқҖм•

    - м»ӨмҠӨн…Җ л§ӨнҠёлҰӯ м¶”к°Җн•ҳм—¬ ActuatorлЎң л°°м№ҳ мғҒнғң мІҙнҒ¬ к°ҖлҠҘн•ҳлҸ„лЎқ кө¬нҳ„

    - мқёкё° лҸ„м„ң н…Ңмқҙлё”м—җ лЎңм§Ғ кІ°кіј м ҖмһҘ кө¬нҳ„

### м„ңнҳ„н•ҳ

- **лҰ¬л·° CRUD л°Ҹ к¶Ңн•ң кІҖмҰқ**
    - мӮ¬мҡ©мһҗк°Җ нҠ№м • лҸ„м„ңм—җ лҢҖн•ҙ лҰ¬л·°лҘј мһ‘м„ұ, мЎ°нҡҢ, мҲҳм •, мӮӯм ңн•  мҲҳ мһҲлҠ” н•өмӢ¬ API кө¬нҳ„
    - JWT кё°л°ҳ мқёмҰқ м •ліҙлҘј нҷңмҡ©н•ҳм—¬ мһ‘м„ұмһҗ ліёмқёл§Ң мҲҳм • л°Ҹ мӮӯм ң лЎңм§Ғм—җ м ‘к·јн•  мҲҳ мһҲлҸ„лЎқ к¶Ңн•ң кІҖмҰқ м Ғмҡ©
- **лҸ„м„ңлі„ лҰ¬л·° мһ‘м„ұ м ңн•ң (л№„мҰҲлӢҲмҠӨ лЎңм§Ғ)**
    - нҠ№м • лҸ„м„ңм—җ лҢҖн•ң лҰ¬л·° лҸ…м җ л°©м§ҖлҘј мң„н•ҙ мӮ¬мҡ©мһҗлӢ№ лҸ„м„ңлі„ 1к°ңмқҳ лҰ¬л·°л§Ң мһ‘м„ұ к°ҖлҠҘн•ҳлҸ„лЎқ м„Өкі„
    - мӨ‘ліө мһ‘м„ұ мӢңлҸ„ мӢң м „м—ӯ м»ӨмҠӨн…Җ мҳҲмҷё(Global Exception) л°ңмғқ л°Ҹ мқјкҙҖлҗң м—җлҹ¬ мқ‘лӢө мІҳлҰ¬
- **мЎ°нҡҢ м„ұлҠҘ мөңм Ғнҷ”**
    - л§Ҳм§Җл§үмңјлЎң мЎ°нҡҢн•ң мӢқлі„мһҗ кё°мӨҖмқҳ м»Өм„ң кё°л°ҳ нҺҳмқҙм§Җл„Өмқҙм…ҳ лҸ„мһ…мңјлЎң DB мЎ°нҡҢ м„ұлҠҘ мөңм Ғнҷ”
- **ліөн•© лҚ°мқҙн„° лӢЁмқј мқ‘лӢө мІҳлҰ¬**
    - лӘ©лЎқ мЎ°нҡҢ мӢң нҳ„мһ¬ лЎңк·ёмқён•ң мӮ¬мҡ©мһҗмқҳ мўӢм•„мҡ” м—¬л¶ҖлҘј мқ‘лӢөм—җ нҸ¬н•Ён•ҳм—¬ н”„лЎ нҠём—”л“ң API нҳём¶ң мөңмҶҢнҷ”
    - N+1 л¬ём ң л°©м§ҖлҘј мң„н•ң мҝјлҰ¬ мөңм Ғнҷ” мҲҳн–ү

### мңӨкёҲл№„

- **мӮ¬мҡ©мһҗ нҡҢмӣҗк°Җмһ… л°Ҹ лЎңк·ёмқё**
    - Stateless мқёмҰқ кө¬мЎ°: м„ём…ҳ/нҶ нҒ° м ҖмһҘмҶҢ м—Ҷмқҙ кі мң  мӢқлі„мһҗлҘј л°ҳнҷҳн•ҳм—¬ нҒҙлқјмқҙм–ёнҠё мӨ‘мӢ¬мқҳ мғҒнғң кҙҖлҰ¬ кө¬нҳ„
    - ліҙм•Ҳ л°Ҹ мң нҡЁм„ұ кІҖмҰқ: BCrypt м•”нҳёнҷ” л°Ҹ м •к·ңн‘ңнҳ„мӢқ кё°л°ҳ мң нҡЁм„ұ кІҖмҰқ м–ҙл…ён…Ңмқҙм…ҳ м Ғмҡ©
    - мһ¬к°Җмһ… л°©м§Җ лЎңм§Ғ: л…јлҰ¬ мӮӯм ңлҗң кі„м •мқ„ нҸ¬н•Ён•ң м „мІҙ мқҙл©”мқј мӨ‘ліө кІҖмӮ¬лЎң нғҲнҮҙ кі„м • мһ¬к°Җмһ… мӣҗмІң м°ЁлӢЁ
    - API л¬ём„ң мөңм Ғнҷ”: Swagger лӮҙ @SchemaлҘј нҷңмҡ©н•ҳм—¬ м •к·ңмӢқ кё°л°ҳ лӮңмҲҳ мғқм„ұ мҳӨлҘҳ н•ҙкІ° л°Ҹ мҳҲмӢңк°’ кі м •
- **мӮ¬мҡ©мһҗ м •ліҙ кҙҖлҰ¬ л°Ҹ ліҙм•Ҳ**
    - лҰ¬мҶҢмҠӨ мҶҢмң к¶Ң кІҖмҰқ: мҡ”мІӯ н—ӨлҚ” IDмҷҖ лҢҖмғҒ лҰ¬мҶҢмҠӨ ID лҢҖмЎ°лҘј нҶөн•ң нғҖмқё м •ліҙ мҲҳм • л°Ҹ м ‘к·ј м°ЁлӢЁ
- **мӮ¬мҡ©мһҗ л…јлҰ¬ л°Ҹ л¬јлҰ¬ мӮӯм ң **
    - мқҙмӨ‘ мӮӯм ң м „лһө: BaseFullAuditEntity кё°л°ҳ л…јлҰ¬ мӮӯм ңмҷҖ л°°м№ҳ мІҳлҰ¬лҘј нҶөн•ң л¬јлҰ¬ мӮӯм ң лі‘н–ү
    - Spring Batch мһҗлҸҷнҷ”: л§Өмқј мһҗм • л…јлҰ¬ мӮӯм ң нӣ„ 24мӢңк°„ кІҪкіј лҚ°мқҙн„°лҘј мһҗлҸҷ м ңкұ°н•ҳлҠ” л°°м№ҳ нҢҢмқҙн”„лқјмқё кө¬м¶•
    - мһ‘м—… м•Ҳм •м„ұ нҷ•ліҙ: JobParameters кё°л°ҳ мӨ‘ліө мӢӨн–ү л°©м§Җ л°Ҹ лӢЁкі„лі„(Job-Step-Tasklet) кө¬мЎ° м„Өкі„
- **лҚ°мқҙн„° м •н•©м„ұ л°Ҹ лӘЁлӢҲн„°л§Ғ**
    - м—°мҮ„ мӮӯм ң мһҗлҸҷнҷ”: DB л ҲлІЁ ON DELETE CASCADE м„Өм •мқ„ нҶөн•ң кҙҖл Ё лҚ°мқҙн„° м—°кі„ л¬ҙкІ°м„ұ ліҙмһҘ
